PT. TRIGO PRIMA UNIVERSAL - WEBSITE

FILE:
- header.html      -> Navigasi yang dipanggil oleh semua halaman
- style.css        -> Semua CSS utama
- index.html       -> Homepage
- about.html       -> Profil / About Us
- programs.html    -> Program & Layanan
- news.html        -> Berita (placeholder)
- contact.html     -> Contact Us

CARA MENJALANKAN:
1. Ekstrak semua file ke satu folder.
2. Pastikan ketujuh file berada di folder yang sama.
3. Buka index.html melalui Live Server (disarankan VS Code).

CATATAN:
- Header dipanggil menggunakan fetch('header.html').
- Karena itu, beberapa browser dapat memblokirnya jika index.html dibuka langsung melalui file://.
- Gunakan VS Code + ekstensi Live Server, atau upload ke hosting.
- Chart.js digunakan dari CDN pada homepage.
- Ganti nomor telepon, email, alamat, foto, dan data statistik dengan data asli perusahaan.
